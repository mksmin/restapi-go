# RestAPIGo

Учебный проект по созданию RESTful API на языке Go для управления списком задач (ToDo list). Проект демонстрирует работу с базой данных PostgreSQL, использование SQLX и организацию структуры проекта по стандартам Go.

## 🚀 Стек технологий

*   **Language:** Go (Golang)
*   **Database:** PostgreSQL
*   **Libraries:**
    *   `sqlx` — расширение для стандартного `database/sql` для удобной работы с БД.
    *   `lib/pq` — драйвер PostgreSQL.
    *   `net/http` — стандартная библиотека для создания API.

## 📂 Структура проекта

```text
.
├── cmd/api/            # Точка входа в приложение
│   └── main.go         # Инициализация сервера и роутинга
├── internal/
│   ├── database/       # Логика работы с базой данных
│   │   ├── database.go # Подключение к БД
│   │   └── tasks.go    # CRUD операции для задач
│   ├── handlers/       # HTTP обработчики (контроллеры)
│   └── models/         # Описание структур данных (D TO)
├── sql/                # SQL скрипты
│   └── init.sql        # Скрипт инициализации таблиц
└── go.mod              # Зависимости проекта
```

## 🛠 Установка и запуск

### 1. Требования
*   Установленный Go (1.18+)
*   Установленный PostgreSQL

### 2. Настройка базы данных
Создайте базу данных и выполните скрипт инициализации:
```bash
psql -U postgres -d your_db_name -f sql/init.sql
```

### 3. Запуск приложения
Склонируйте репозиторий и запустите сервер:
```bash
go run cmd/api/main.go
```
Сервер запустится на порту `:8083` (по умолчанию).

## 📡 API Эндпоинты

| Метод  | Эндпоинт           | Описание                    |
| :----- | :----------------- | :-------------------------- |
| GET    | `/tasks`           | Получить список всех задач  |
| POST   | `/tasks/create`    | Создать новую задачу        |
| GET    | `/tasks/{id}`      | Получить задачу по ID       |
| PUT    | `/tasks/{id}`      | Обновить задачу по ID       |
| DELETE | `/tasks/{id}`      | Удалить задачу по ID        |

### Примеры запросов

**Создание задачи (POST):**
```bash
curl -X POST http://localhost:8083/tasks/create \
     -H "Content-Type: application/json" \
     -d '{"title":"Изучить Go", "description":"Пройти курс по стандартной библиотеке", "completed":false}'
```

**Получение всех задач (GET):**
```bash
curl http://localhost:8083/tasks
```
